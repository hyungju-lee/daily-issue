Markup components guide

  • js-{layout} class는 css selector로 사용하지 않습니다.
  • js-프리픽스가 붙지 않은 class는 디자인에 맞춰 변경하여 사용 가능합니다.
  • 프리픽스 적용된 구조가 유지될 수 있도록 유의하여 사용해주세요.

아코디언
기본 구조

-.js-accordion-list 아코디언 wrapper

-.js-accordion-item 아코디언 탭&컨텐츠를 감싸는 영역

-.js-accordion-tab 아코디언 탭. aria-controls="{컨텐츠ID}" 속성 적용

-.js-accordion-content 아코디언 컨텐츠. 탭 aria-controls 속성과 연결된 id 적용

- 탭-컨텐츠 연결은 index 기준
